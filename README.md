# JSON Codo theme

Output json for classes, mixins, and extra files.

## Usage

First

    $ npm install codo-theme-json --save

Then, add `--theme json` to `.codoopts` or command line.

## Output

    ├── class
    |   ├── ...json files
    |   ├── index.json
    |   |
    ├── mixin
    |   ├── ...json files
    |   ├── index.json
    ├── extra
    |   ├── ...json files
    |   ├── index.json
